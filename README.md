# AI Personalized Learning Chatbot

Full-stack application with React frontend and FastAPI backend for adaptive learning.

## Features
- AI chatbot tutor with adaptive difficulty
- Quiz generation system
- Student progress tracking
- Gamification (points, badges, leaderboard)
- Text and voice interaction support
- REST API architecture

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Access at http://localhost:3000
